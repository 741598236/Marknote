# 👋 欢迎使用 MarkNote

欢迎来到 MarkNote，一款简洁高效的 Markdown 笔记应用！以下是快速入门指南：

***

## 🚀 基本功能

1. **实时预览**：左侧编辑，右侧即时呈现排版效果
2. **文件管理**：通过侧边栏轻松创建/删除笔记
3. **自动保存**：所有更改都会自动保存
4. **跨平台**：支持 Windows/macOS/Linux

***

## 📝 Markdown 快速参考

### 标题

```javascript
# 一级标题

## 二级标题

### 三级标题
```

### 文本样式

```markdown
_斜体_ 或 _斜体_  
**粗体** 或 **粗体**

```

## 💡 小技巧

1. 使用 `Tab` 键缩进代码块
2. 输入 `---` 自动生成分割线
3. 拖放图片到编辑器可直接插入
