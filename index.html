优雅的 Markdown 编辑器

优雅的 Markdown 编辑器

欢迎来到 MarkNote！这是一个为你量身打造的 Markdown 笔记应用。

“简洁、高效、专注”

✨ 功能特性

  • 实时 Markdown 预览：即时呈现排版效果
  • 简洁文件管理：通过标签和智能分组轻松管理笔记
  • 自动保存：所有更改自动保存，无需手动操作
  • 跨平台支持：Windows/macOS/Linux 全平台兼容
  • 主题切换：内置昼夜主题，保护您的眼睛
  • 代码高亮：支持多种编程语言语法高亮
  • 数学公式支持： 使用 LaTeX 语法轻松编写公式，例如：$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$
  • 图表绘制： 支持 Mermaid 语法绘制流程图、序列图等：
graph TD;
    A[开始] --> B{决策};
    B -- 是 --> C[行动];
    C --> D[结束];
    B -- 否 --> E[等待];
    E --> A;

🛠️ 技术栈

  • 前端框架: React + TypeScript
  • 构建工具: Vite
  • UI 框架: TailwindCSS
  • 编辑器: CodeMirror
  • 打包工具: Electron + electron-builder

🚀 快速开始

# 克隆项目
git clone https://gitee.com/your-repo/marknote.git

# 安装依赖
yarn install

# 启动开发模式
yarn dev
